# Operators

Grid provides a variety of operators for use in expressions, with some particular interactions between different types.

## Relational

```
< <= != == >= >
```

*int, float, char (ordinal), str (length)*

## Boolean

```
&& || !!
```

*Relational, bool*

## Bitwise

```
& | ^ ! << >>
```

*int, char*

## Arithmetic
```
+ - * / % **
```

*int, float*

## Assignment
```
+= -= *= /=
```

*int, float*
